# KantoTrainApi

A RESTful API for KantoTrain - https://kanto-train-api.herokuapp.com/

# Setup

```bash
$ git clone https://github.com/KantoTrain/KantoTrainApi
$ cd KantoTrainApi
$ bundle install
$ bundle exec rackup
```

Browse http://localhost:9292/
